Laronix Naturalness Test

Thanks for participating Laronix's speech naturalness test.
This test will last for 15 to 20 minutes.
test contains 3 sets, you can select one set or finish all of them

Instruction

  1. Input you name and select one set, then click "Start"
  2. Listen to the utterance, and select the most proper evaluation of naturalness among 5 choices.
  3. There are 5 sentences per page, after evaluation you will see the "Next" button, Click that to continue.
  4. After finishing all the evaluation you will see the "Finish" Button。Click it, and a file named "name"_set_n.csv will be automatically downloaded.
  5. Send this file to

    6. testemail@testemail.com

About naturalness evaluation(※Please read this carefully!)

Naturalness

The naturalness should be evaluated by how similar it sound comparing to natural human voice, rather than how much you can dictate, that represents intelligibility.
So, if you think the sound is almost generated by human's larynx, please score it with high marks.
