Laronix Naturalness Test

Thank you for participating Laronix's speech naturalness test.

  • This test will last for 15 to 20 minutes.
  • Test contains 5 sets, you can select one set or finish all of them.

Instruction

  1. Input you name and select one set, then click start.
  2. Listen to the audio, and select the most proper evaluation of naturalness among 5 choices.
    • Excellent
    • Good
    • Fair
    • Poor
    • Bad
  3. There are 5 sentences per page, after evaluation you will see the next button, Click that to continue.
  4. After finishing all the evaluation you will see the finish Button。Click it, and a file (your_name)_set_(n).csv will be automatically downloaded.
  5. Send this file to

    6. testemail@testemail.com

Miscellaneous

  • Please participate with PC rather than mobile phone.
  • Using Headphones or speaker under a quite environment.
  • Once the test start, don't close or refresh the page until it's done.
  • If the audio can not be played, select Fair and report to us.

What is naturalness？ (※Please read this carefully!)

The naturalness should be evaluated by how similar it sound comparing to natural human voice, rather than how much you can dictate, that represents intelligibility.
So, if you think the sound is most likely generated by human's larynx, please score it with high marks.
